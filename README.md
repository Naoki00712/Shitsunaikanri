# 室内管理アプリ(学習用リポジトリテスト)

ラズパイでDHT11とBMP280を扱った
温度湿度気圧を測定して屋外（神戸市）の気温湿度気圧を測定するプログラムです。
Ambientで測定したデータをグラフ化しているので
CHANNELID = ''
WRITEKEY = ''
この二つの部分を変えて扱ってください。
